# Odream App

Odream App은 adc에서 진행하는 부트캠프 프로젝트로 KREAM(https://kream.co.kr/) 모바일 웹 클론 앱입니다.
... 는 아직 작업중🚧

## 사용된 기술 스택

- NextJS
- Typescript
- Redux toolkit
- React-query
- Next-auth
- Axios
- Jest
- pnpm
- Tailwind CSS
